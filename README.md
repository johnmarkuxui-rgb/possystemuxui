## ✨ Key Features

- 📦 **Real-time Inventory:** Dynamic stock monitoring and management with instant updates.
- 🛡️ **Manager Authorization:** Secure terminal access with multi-level administrative overrides.
- ₱ **Localized Pricing:** Optimized for **PHP (Philippine Peso)** currency and local business tax logic.
- ⚡ **Offline-First Persistence:** Zero-latency data handling utilizing local persistence for maximum speed.
- 📱 **Responsive Design:** Fully adaptive UI/UX for both desktop terminals and mobile devices.

---

## 🔄 CI/CD Workflow

1. **Development:** Local iteration using **Trae IDE** & **Next.js**.
2. **Version Control:** Managed via **GitHub** with semantic commits.
3. **Deployment:** Automated builds and global edge hosting via **Vercel**.
